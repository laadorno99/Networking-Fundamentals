# Common Network Protocols and Ports

A reference list of essential protocols used in network communication and security.

| Protocol | Port | Description |
| --- | --- | --- |
| HTTP | 80 | Hypertext Transfer Protocol (Unsecure) |
| HTTPS | 443 | Secure HTTP (TLS/SSL) |
| SSH | 22 | Secure Shell (Remote access) |
| DNS | 53 | Domain Name System |
| FTP | 21 | File Transfer Protocol |
| DHCP | 67/68 | Dynamic Host Configuration Protocol |
