# OSI Model Summary

Understanding the 7 layers of the OSI model is essential for network troubleshooting and security analysis.

### Layers Breakdown:
1. **Physical:** Transmission of raw bit streams over a physical medium.
2. **Data Link:** Reliable transmission of data frames between two nodes (MAC addresses).
3. **Network:** Routing and addressing (IP addresses, ICMP).
4. **Transport:** End-to-end communication and error recovery (TCP/UDP).
5. **Session:** Management of sessions between applications.
6. **Presentation:** Data representation, encryption, and decryption.
7. **Application:** Network process to application (HTTP, FTP, DNS).
