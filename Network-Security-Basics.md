# Network Security Fundamentals

Key concepts studied during my initial certifications.

## Essential Security Devices
* **Firewall:** Acts as a barrier between a trusted network and an untrusted network (like the internet).
* **Security Fabric:** A concept from Fortinet where different security devices work together to share information.

## Basic Protection
* **Encryption:** Protecting data so that only authorized people can read it.
* **Access Control:** Making sure only the right people can enter specific parts of the network.
